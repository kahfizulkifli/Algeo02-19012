Konsep singkat query

Ketika search engine menerima query, search engine akan membersihkan string query dan mengubahnya menjadi array of terms. Selain itu, search engine juga akan membersihkan tiap dokumen yang ada di folder test. Lalu, untuk setiap dokumen di folder test, search engine akan menghitung tingkat kemiripannya dengan query dengan menggunakan konsep cosine similarity. Setelah itu, informasi tentang masing-masing dokumen diurutkan berdasarkan tingkat kemiripan dan ditampilkan pada laman web.

How to use

Mengunggah dokumen

  1. Klik tombol "Choose File" pada bagian atas laman web.
  2. Pilih dokumen yang hendak diunggah, dokumen harus merupakan file .txt.
  3. Klik tombol "Submit".

Mencari dokumen

  1. Masukkan query pada text box pada bagian atas laman web.
  2. Klik tombol "Submit".

Membuka dokumen

Klik judul dokumen.
